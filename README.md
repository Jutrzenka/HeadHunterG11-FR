## :dragon_face: Installation project

**Clone the project:**
```
git clone https://github.com/Jutrzenka/HeadHunterG11-FR.git
```
**Go to the project directory:**
```
cd HeadHunterG11-FR
```
**Install dependencies:**
```
npm install
```
**Start the server:**
```
npm start
```
